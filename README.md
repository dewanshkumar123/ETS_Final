# Lecture Companion Prototype

A lightweight web prototype built with Flask, HTML, CSS, and JavaScript to explore how classroom learning can be enhanced using technology.

The system allows teachers to create lectures from transcripts and automatically generate formatted notes. Students can view course lectures and ask follow up questions through a simple chatbot. Authentication, role based access, and JSON storage are included.

---

## Features

- Login and signup system
- Teacher dashboard to create lectures with title, course name, and transcript
- Automatic basic note generation
- Student view filtered by course
- Built in chatbot placeholder for Q&A
